About this prototype

A fairness-weighted heuristic over demo data for a 32-bed unit — honors picks, balances weekends, protects PTO and orientee→preceptor pairing, and flags what it can't satisfy. Production adds Kronos/MyTime import, a real constraint solver, and cloud storage so ledgers persist across cycles. Edits live in memory for this session.
